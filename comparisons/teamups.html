组合推荐与协同逻辑

双人组合

  • 游侠 + 盗贼：游侠提供 易伤 与远程压血，盗贼利用 减益 和残血阈值完成高速收割。
  • 奥术学者 + 炎术师：奥术学者负责聚怪和重排站位，炎术师负责火场覆盖和范围引爆。
  • 吟游诗人 + 盗贼：吟游诗人用换位和音符提高 盗贼的切入、撤离与回合效率。
  • 先锋 + 炎术师：先锋通过 跳跃 和 击退 把敌人压进更利于 范围效果 的区域，方便 炎术师扩大火场价值。
  • 狂战士 + 游侠：狂战士顶在前排拉仇恨，游侠在安全距离持续打 精准 和 易伤。

三人组合

  • 狂战士 + 游侠 + 盗贼：标准“前排顶住 + 远程挂易伤 + 刺客收割”组合，稳定且容易理解。
  • 奥术学者 + 炎术师 + 吟游诗人：控场、铺火、团队节奏同时在线，适合追求地图操作感和连段感的队伍。
  • 先锋 + 游侠 + 吟游诗人：先锋主动开团，游侠稳定点杀，吟游诗人负责补机动和团队续航，推进节奏非常顺。
  • 狂战士 + 炎术师 + 吟游诗人：近战区更稳，炎术师有更好的怪物密度和保护，吟游诗人则持续补位移和护盾。

四人组合

  • 狂战士 + 游侠 + 盗贼 + 吟游诗人：经典稳健队。狂战士顶前排，游侠负责稳定挂易伤，盗贼完成切后排和收割，吟游诗人补换位、护盾和团队节奏，适合大多数常规推进关卡。
  • 奥术学者 + 炎术师 + 先锋 + 吟游诗人：偏控场开团队。先锋负责主动开团，奥术学者负责重排站位和聚怪，炎术师接火场与范围爆发，吟游诗人补机动与保底，适合怪群密度高、地形可利用的战斗。
  • 狂战士 + 奥术学者 + 炎术师 + 游侠：偏中远程火力队。狂战士稳前线，奥术学者负责控位与救援，炎术师处理范围压血，游侠负责远程点杀和持续易伤，适合需要同时兼顾前排稳定与后排输出的队伍。
  • 先锋 + 盗贼 + 游侠 + 吟游诗人：偏高机动节奏队。先锋先手开路，盗贼抓落单与收残局，游侠在后方稳定点杀，吟游诗人负责全队换位与资源调度，适合追求主动节奏和连续击杀的玩法。

机制拆分与相关角色

  • 地面物件：相关角色有吟游诗人的音符、炎术师的火焰、游侠的藤蔓、盗贼可互动后的金币 / 战利品。相关价值在于铺场、封路、转化、踩物件取收益，以及把地面资源变成治疗、力量或范围压制手段。
  • 易伤：相关角色以游侠为核心，代表技能包括“穿刺箭”“前哨侦行”，狂战士的近身压制也能帮助队友更稳定吃到易伤后的集火收益。相关价值在于提高全队后续点杀和收割效率。
  • 虚弱：相关角色包括狂战士、盗贼、先锋。代表技能包括狂战士的“尽管放马过来”、盗贼的“千刃连袭”、先锋的“英雄时刻”与终极技。相关价值在于削弱怪物回合质量，降低前排承伤压力，并为中距离换血创造安全空间。
  • 位移与传送：相关角色包括奥术学者、吟游诗人、先锋、盗贼。奥术学者负责传送与站位重排，吟游诗人负责换位与魅惑调度，先锋负责跳跃进场和击退，盗贼负责高机动切入。相关价值在于快速重构战场、制造孤立目标和优化队友落位。
  • 火场与范围压制：相关角色以炎术师为核心，奥术学者和先锋适合补聚怪与压位。相关价值在于先把怪物逼入高密度区域，再用火焰、引爆与范围技能放大群体伤害。
  • 潜幕与收割：相关角色以盗贼为核心，游侠和奥术学者常作为辅助。游侠负责远程压血与挂易伤，奥术学者负责控位与聚怪，盗贼则在残局用潜幕切入完成单体斩杀。相关价值在于把队伍前半段制造的血线优势转成稳定收头。
  • 诱饵与分身：相关角色主要是奥术学者与盗贼。奥术学者能用诱饵做分身联动与空间干扰，盗贼能用诱饵制造致盲后的错位压力。相关价值在于吸引目标、扰乱站位、补足额外伤害和增加技能释放窗口。
